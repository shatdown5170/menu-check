Feature 01 · 02

사진 업로드 & AI 자동 분석

다양한 방법으로 메뉴판 이미지를 입력받아 AI가 즉시 구조화된 데이터로 변환
📷

카메라 즉시 촬영

모바일 기기의 카메라를 직접 실행해 현장에서 바로 메뉴판을 촬영·업로드

모바일 전용
🖼️

갤러리 이미지 선택

기존에 촬영해둔 이미지를 갤러리에서 선택. 다중 파일 동시 선택 지원으로 여러 장의 메뉴판을 한 번에 처리

다중 선택
🖱️

PC 드래그앤드롭

PC 환경에서 파일 탐색기의 이미지를 웹 화면으로 끌어다 놓으면 즉시 업로드

PC 지원
🤖

AI OCR + 자동 구조화

  • 메뉴명·가격·설명·옵션 자동 추출
  • 카테고리 자동 분류 (예: 메인, 사이드, 음료)
  • 판독 불가 항목 별도 플래그 처리
  • 다중 이미지 병렬 분석
🖼️

업로드 이미지 미리보기

업로드한 원본 사진을 썸네일로 확인. 탭 전환으로 여러 장 비교 가능하며 팝업으로 원본 확대 열람

⚠️

품질 경고 표시

AI가 판독하기 어려운 항목(흐린 사진, 손글씨 등)을 자동 감지해 검수자에게 경고 배너로 안내

리스크 감소

Feature 03 · 04

메뉴 검수 & 편집

AI 초안을 담당자가 직접 수정·보완하여 최종 데이터를 완성하는 인라인 편집 워크플로우
✏️

인라인 항목 편집

  • 메뉴명 직접 수정
  • 가격·가격 표기 방식 선택
  • 메뉴 설명 텍스트 편집
  • 옵션(예: 사이즈, 맛) 추가·삭제
  • 이미지 URL 연결
🏷️

상태 태그 관리

  • 품절 표시 토글
  • 메뉴판 미노출 처리 (숨김)
  • 대표 메뉴 지정
  • 신메뉴 플래그
➕

항목 추가 / 삭제

AI가 인식하지 못한 메뉴를 수동으로 추가하거나, 불필요한 항목을 즉시 삭제. 항목별 개별 확장·접기 지원

📂

카테고리 탭 네비게이션

AI가 분류한 카테고리를 상단 탭으로 표시. 스크롤 연동으로 현재 위치 자동 하이라이트. 미분류 항목은 '미맵핑된 디쉬' 탭으로 별도 관리

⠿

드래그앤드롭 순서 변경

  • 컴팩트 카드 모드로 전환 (이름·가격만 표시)
  • Ghost 클론 애니메이션으로 부드러운 이동
  • 주변 카드 실시간 translateY 반응
  • 모바일 터치·PC 포인터 동일 동작
터치 지원
🔍

이미지 팝업 확대

각 메뉴 항목의 썸네일을 탭하면 원본 이미지를 팝업으로 확대 표시. contain 모드로 전체 이미지를 왜곡 없이 확인

Comparison

도입 전 · 후 비교

기존 수작업 프로세스 대비 AI 도입 후 변화 항목
항목 도입 전 (수작업) 도입 후 (AI 자동화)
데이터 입력 메뉴판을 보며 항목 하나씩 수작업 타이핑 사진 업로드 → AI 초안 자동 생성 → 검수만 수행
오류 발생 오탈자·가격 오류·항목 누락 상존, 사람이 직접 대조·검수 AI 1차 전처리로 오류 사전 감지, 검수자는 예외만 처리
처리 속도 메뉴 1건당 수십 초, 전체 메뉴판 처리에 수십 분 사진 1장 분석 수 초, 다중 사진 병렬 처리
판독 어려운 글씨 담당자가 직접 확대·해석 후 입력 AI 자동 감지 후 경고 표시, 담당자에게 집중 안내
카테고리 분류 분류 기준 없이 순서대로 입력 AI 자동 분류 + 미분류 항목 별도 관리
순서 변경 삭제 후 재입력 또는 복사·붙여넣기 드래그앤드롭으로 즉시 재배치 NEW
다중 메뉴판 사진별 개별 작업, 결과 수동 통합 여러 장 동시 업로드 → 자동 통합 결과 생성
※ 정량적 효과(시간 절감율·오류 감소율)는 운영 데이터 축적 후 측정 예정
