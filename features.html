Feature Detail 1

사진 입력 & AI 자동 분석

📷
다중 입력 수단
카메라 촬영, 갤러리 선택, PC 드래그앤드롭 세 가지 방식 모두 지원. 여러 장을 한 번에 업로드하면 AI가 합산 분석.
🔍
OCR + 구조화
Gemini 2.5 Flash가 메뉴판 이미지에서 메뉴명·가격·설명·옵션을 자동 추출. 손으로 쓴 글씨, 복잡한 레이아웃도 인식.
🗂
카테고리 자동 분류
메뉴판 섹션 구조를 파악해 카테고리를 자동 생성. 분류 안 된 항목은 '미맵핑된 디쉬' 그룹으로 별도 표시.
⚡
실시간 진행 표시
분석 중 로딩 상태를 실시간으로 표시. 완료 즉시 편집 화면으로 자동 전환.

Feature Detail 2

검수·편집 & 순서 변경

✏️
항목별 인라인 편집
메뉴명·가격·설명·옵션·품절 여부를 카드 내에서 직접 수정. 별도 팝업 없이 즉시 반영.
🖼️
원본 이미지 대조
각 카드의 썸네일을 탭하면 원본 메뉴판을 팝업으로 확대 표시. 편집 중 원문과 즉시 비교 가능.
↕️
드래그앤드롭 순서 변경
순서 변경 모드 진입 시 컴팩트 카드로 전환. Ghost clone + translateY 애니메이션으로 부드러운 재배치.
➕
수동 항목 추가·삭제
AI가 인식 못한 항목을 수동 추가 가능. 불필요한 항목은 즉시 삭제.
